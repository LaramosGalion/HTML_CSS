Estilos inline

Como visto anteriormente o HTML é uma linguagem focada na semântica, com isso o design foi destinado ao CSS, no primeiro caso a ser estudado será os estilos inline sendo eles estilos aplicados pontualmente no código, dentro das próprias tags que desejam ser alteradas. Eles são muito difíceis de serem aplicados pois agem somente nas tags que eles estão, reafirmando sua ação pontual.
