Estilos externos(external)

Nesse modo de aplicar estilos usasse um arquivo externo(ou pode usar mais arquivos com estilos extras), esse contendo todos os seletores para efetuar a aplicação dos estilos, assim no arquivo HTML usasse a tag 'link:css' na tag 'head', linkando o aquivo, nesse arquivo CSS há regras além dos seletores. DICA - com o ctrl apertado e clicando em cima do arquivo é possivel ser direcionado a ele, mas caso esse arquivo não exista o VS dará a opção de cria-lo. Esse modo de aplicação de estilo é o mais eficiente, é possivel modificar todas as paginas ao mesmo tempo so modificando o arquivo CSS.

É possível misturar os estilos de inserção de estilos no mesmo arquivo, usando inline, internal e external, tendo essa ordem de precedência.
