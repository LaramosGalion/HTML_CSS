Trabalhando Com listas

Listas ordenadas

Listas ordenadas, OL(ordered list), são listas onde é aplicada uma ordem sequência nos itens, li(list item). Ex:

  1. Acordar
  2. Escovar os dentes
  3. Tomar o café da manhã
  4. ...
  5. Lorem, ipsum.
  6. Lorem, ipsum.
  7. Lorem, ipsum.
  8. Lorem, ipsum.
Assim caso necessite alterações é so introduzi-las entre os elementos e ela será rearanjada. Outroa ponto é que o </li> se tornou opicional na versão HTML 5. Os modificadores do parâmetro 'type' são, 1,A,a,I,i, tendo também o parâmetro start, para indicar onde a lista começara.

Listas não ordenadas

Listas não ordenadas, Ul(Unordered list), são listas onde não tem uma distinção da ordem dos elementos da lista. Ex:

  • Acordar
  • Escovar os dentes
  • Tomar o café da manhã
  • ...
  • Lorem, ipsum.
  • Lorem, ipsum.
  • Lorem, ipsum.
  • Lorem, ipsum.
Os modificadores do parâmetro 'type' são: disc, circle, square, não tem parâmetro start

Listas Mistas

Listas mistas são listas que misturam tanto listas ordenadas quando não ordenadas, ou somente um tipo, mas dando niveis para as mesmas. Ex:

  1. Exemplo 1
  2. Exemplo 2
    • Exemplo 1
    • Exemplo 2
    • Exemplo 3
      1. Exemplo 1
      2. Exemplo 2
  3. Exemplo 3
    • Exemplo 1
      1. Exemplo 1

Lista de Definições

HTML
