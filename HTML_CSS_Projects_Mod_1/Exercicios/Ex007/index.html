Sabendo como a semântica funciona no HTML

Semântica tem como objetivo o significado e não a forma, sendo assim o HTML tenta se focar ao maxímo nela, deixando as formas para o CSS. Anteriormente o HTML, na sua versão 4, proporcionava modificações na forma, mas com o tempo a percepção de uso da linguagem foi mudando e agora ela é so destinada a forma em sua maioria.

As evoluções do HTML4 para o HTML5, ocorreram introduções a novas tag, como a 'lang', outras migraram, como 'h1', e outras foram descontinuadas como 'bgcolor'

Tag foram descontinuadas no HTML, agora na versão 5, por darem forma a ele, assim sendo muitas delas ainda podem funcionar para modificar algo visual, mas a maneira mais correta de dar forma é usando a CSS e não usar essas tags, pois em algum momento elas podem parar de funcionar, assim acarretanto em manutenção e modificação de código desnecessario. Para saber quais são as tags que estão obsoletas acesse link: Tags Obsoletas
